input

Wanneer je op een toets drukt ontvangt de computer informatie. Manieren waarop de computer input kan krijgen zijn bijvoorbeeld:met de muis, toetsenboord, touchscreen, controller en webcam.



verwerking

De computer verwerkt de informatie die die heeft gekregen. Proces bestaat uit 3 elementen: Uitvoerbare code(de code moet wel kloppen), Data(de computer heeft informatie nodig om mee te werken.) en uitvoeringscontext.
