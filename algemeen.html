Invoer

Wanneer je op een toets drukt ontvangt de computer informatie. Manieren waarop de computer input kan krijgen zijn bijvoorbeeld:met de muis, toetsenboord, touchscreen, controller en webcam.



Verwerking

De computer verwerkt de informatie die die heeft gekregen. Verwerking bestaat uit 3 dele: Uitvoerbare code(de code moet wel kloppen), Data(de computer heeft deze informatie nodig om er mee te kunnen werken.) en uitvoeringscontext(wat er uiteindelijk gebeurt).



Opslag

Na het verwerken van de code slaat de computer de gekregen informatie op en onthoud de computer deze informatie voor de output



Uitvoer

De computer laat de verwerkte informatie wat in de opslag staat zien op het scherm. Het hoeft niet alleen te zijn dat het op het scherm komt te staan, maar het kan ook zijn bij speakers of grafische rekenmachines.



De werking van alles wat boven is uitgelegd in 1 afbeelding.
